These files manage the low power sleep mode settings and enable essentially zero battery to be used when lid is closed. Copy these files to their respective locations and then follow up to and including Step 3 of this guide https://ubuntuhandbook.org/index.php/2021/08/enable-hibernate-ubuntu-21-10/ 

logind.conf -> /etc/systemd/logind.conf
sleep.conf -> /etc/systemd/sleep.conf
